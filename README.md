```markdown
# MixGanha — Template inicial

Site de afiliado híbrido (cursos, SaaS e produtos físicos) usando Jekyll e pronto para GitHub Pages.

O que tem aqui:
- Template mínimo Jekyll (layouts, includes)
- Include para links de afiliado com UTM automático
- Include para tabela comparativa
- Página inicial e um post de review de exemplo
- Exemplo de redirect simples em /go/

Como usar:
1. Crie (ou use) o repositório no GitHub (ex.: `mixganha`).
2. Copie estes arquivos para a raiz do repo (respeite pastas: `_layouts`, `_includes`, `_posts`, `go/`).
3. Commit & push:
   - git add .
   - git commit -m "inicial: MixGanha"
   - git push origin main
4. Ative GitHub Pages (Settings > Pages) ou use `username.github.io`.
5. Configure Google Analytics / Search Console conforme preferir.
6. Edite `_config.yml` com seu domínio/autor e comece a publicar posts em `_posts/`.

Divulgação legal:
Este site contém links de afiliado; todas as recomendações devem ter disclosure.
```
